Event propagation

We click in p element nested in div element, and the onclick callback inside the parent element get called

This is bubbling!

When an event happens on an element, first runs the handler on that element, then the parent, then all the way to highest ancestors

Clicking the following button, will cause the callback of four element to get called.

This is an example of capturing, when you click on p element, instead of calling the call back of the p element then the div element, first the div element get handled then the p element.
